## 标题

在一段文本前面加 `#` 号，可以将其转换为标题。

如下表所示：

| Markdown 语法  |     渲染效果      |
| :------------: | :---------------: |
|  `# 一级标题`  | <h1>一级标题</h1> |
| `## 二级标题`  | <h2>二级标题</h2> |
| `### 三级标题` | <h3>三级标题</h3> |

标题的级别由 `#` 号的数量决定，一级标题对应一个 `#` 号，二级标题对应两个 `#` 号，以此类推，最多支持六级标题。

::alert{type="info"}
不同的编辑器处理 `#` 号的方式不同，有的编辑器需要在 `#` 号后面加一个空格，有的编辑器则不需要。 :br
:br
为了兼容性考虑，请使用空格（`Space`）将 `#` 号与标题文本分开。
::


## 段落

与传统的文本编辑软件不同，Markdown 中的段落并不是由空行分隔的，而是由一个或多个空行隔开的。

如下表所示：

|              Markdown 语法              |            渲染效果             |
| :-------------------------------------: | :-----------------------------: |
| `这是一个段落` :br :br `这是另一个段落` | 这是一个段落 :br 这是另一个段落 |

::alert{type="info"}
在 *Markdown* 中，一般情况下不使用空格（`Space`）或制表符（`Tab`）来缩进段落，所有文本均顶格书写。
::


## 强调

*Markdown* 使用星号（`*`）和下划线（`_`）作为标记强调字词的符号。

### 斜体（Italic）

使用一个星号（`*`）或下划线（`_`）将文本包围起来，可以用斜体来突出显示文本。

例如：

| Markdown 语法 |   渲染效果    |
| :-----------: | :-----------: |
|   `*斜体*`    | <em>斜体</em> |
|   `_斜体_`    | <em>斜体</em> |


### 粗体（Bold）

使用两个星号（`**`）或下划线（`__`）将文本包围起来，可以用粗体来突出显示文本。

例如：

| Markdown 语法 |       渲染效果        |
| :-----------: | :-------------------: |
|  `**粗体**`   | <strong>粗体</strong> |
|  `__粗体__`   | <strong>粗体</strong> |


### 粗斜体（Bold & Italic）

使用三个星号（`***`）或下划线（`___`）将文本包围起来，可以同时使用粗体和斜体来突出显示文本。

例如：

| Markdown 语法  |             渲染效果             |
| :------------: | :------------------------------: |
| `***粗斜体***` | <strong><em>粗斜体</em></strong> |
| `___粗斜体___` | <strong><em>粗斜体</em></strong> |

::alert{type="info"}
由于 *Markdown* 编辑器之间的差异，星号（`*`）和下划线（`_`）的渲染效果可能不同，所以人们通常使用星号（`*`）来作为强调的标记符号。
::

## 链接

*Markdown* 中的链接语法非常简单，只需将链接地址放在尖括号（`<>`）中或是直接在新的一行添加链接即可构成可点击的链接。

如下表所示：

|        Markdown 语法         |          渲染效果          |
| :--------------------------: | :------------------------: |
|  `https://shift2modern.dev`  |  https://shift2modern.dev  |
| `<https://shift2modern.dev>` | <https://shift2modern.dev> |
|     `<mail@example.com>`     |     <mail@example.com>     |

::alert{type="info"}
尖括号（`<>`）中的链接地址可以是任意的 *URL*，包括 `http://`、`https://`、`ftp://` 等协议开头的链接，也可以是电子邮件地址。
::

### 标准链接

更标准的用法是将链接文本放在方括号（`[]`）中，将链接地址放在小括号（`()`）中，就像这样：

`[Shift2Modern](https://shift2modern.dev)` 

渲染效果如下:

[Shift2Modern](https://shift2modern.dev)

如果想给链接添加一个标题，可以在链接地址后面添加一个空格，然后在用双引号（`""`）将标题包围起来，就像这样：

`[Shift2Modern](https://shift2modern.dev "Shift2Modern")`

将鼠标悬停在链接上，会显示 `Shift2Modern` 这个标题。

[Shift2Modern](https://shift2modern.dev "Shift2Modern")

::alert{type="info"}
不同 *Markdown* 编辑器处理链接中空格的方式不同，所以人们习惯于用`%20`来代替空格。 :br
例如，`https://shift2modern.dev/Shift2Modern%20Blog` 与 `https://shift2modern.dev/Shift2Modern Blog` 是等价的。
::


### 链接的修饰

链接是可以格式化的，可以使用强调、代码等语法来修饰链接文本。 :br
在链接前后增加星号（`*`）可以将链接转换为斜体或粗体。 :br
如需将链接表示为代码，请在方括号（`[]`）前后添加反引号（`` ` ``）。



## 图片

*Markdown* 中的图片语法与链接语法非常相似，只需在[标准链接语法](#标准链接)的基础上在前面加上一个感叹号 `!` 即可。

例如：

`![Markdown-logo](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/1280px-Markdown-mark.svg.png)`

渲染效果如下：

<img  src=https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/1280px-Markdown-mark.svg.png title="Markdown-logo" width=200 height=200/>


::alert{type="warning"}
*Markdown* 原生不支持图片大小调节，如有需要请使用 *HTML* 中的[方法](https://www.w3school.com.cn/tags/att_img_height-width.asp) 。
::

如需给图片增加可点击链接，只需要将**整个**图片代码放在方括号（`[]`）中，将链接放在小括号（`()`）中即可。

就像这样：
`[![图片](图片链接)](链接)`

例如：

`[![Markdown-logo](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/1280px-Markdown-mark.svg.png)](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/1280px-Markdown-mark.svg.png)`

渲染效果如下：

[<img  src=https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/1280px-Markdown-mark.svg.png title="Markdown-logo" width=200 height=200/>](https://upload.wikimedia.org/wikipedia/commons/thumb/4/48/Markdown-mark.svg/1280px-Markdown-mark.svg.png)
